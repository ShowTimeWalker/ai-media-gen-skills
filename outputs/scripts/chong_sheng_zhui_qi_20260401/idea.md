# 重生追妻

> 创建日期：2026-04-01

## 创意核心

前世为了利益抛弃了真心爱自己的女人，重生后她已嫁作人妇、满身伤痕，他拼尽全力也要把她从泥潭里拉出来，哪怕要跟全世界为敌。

## 类型方向

- **类型**：重生 + 追妻 + 甜宠
- **频向**：男频

## 核心情绪

前期解气（打脸渣男、清理障碍），后期甜暖（真心守护、双向奔赴）

## 创意素材

- **男主身份**：商人，前世为了利益放弃了真心
- **前世过错**：为了商业利益放弃了真心爱自己的女人，选择了另一段"有利可图"的关系
- **女主现状**：已嫁人，不再信任男主，被家暴，满身伤痕
- **家暴老公**：纯粹渣男反派，小卡拉米，第一章就下线——节奏拉满
- **女主家人**：也是追妻的阻碍，反对两人复合
- **情感弧线**：悔悟 → 赎罪 → 追妻 → 打破信任壁垒 → 甜宠升温
- **目标观众感受**：前期看男主清理障碍大快人心，后期看真心守护温暖治愈

## 参考类型

**男频重生+权谋**的核心机制可复用：
- 利用前世信息差进行精准操作（商人身份的天然优势：知道市场走向、竞争对手底牌）
- 压抑蓄力 → 小胜积攒信心 → 大爆发，爽感递增的节奏

**女频虐恋+追妻**的情绪节奏可借鉴：
- 男主追妻的方式要具体有诚意，不能只是"花钱摆平"
- 女主清醒、不再忍——这个特质保留，让女主有独立人格
- 信任重建是核心难点，不能"道个歉就回去"

**甜宠+霸总（新派）**的甜宠要素：
- 男主对女主的偏爱和独宠要具体、不油腻
- 女主不是白莲花，有自己的态度和判断
