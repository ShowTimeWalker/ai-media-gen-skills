# 小说封面生图最佳实践

基于 Seedream 5.0（豆包文生图）+ 番茄小说封面的实战经验总结。

---

## 一、提示词结构公式

```
[用途声明] + [主体人物描述] + [场景环境] + [风格美学] + [标题文字要求]
```

每部分用自然语言连贯描述，不要用逗号分隔的关键词堆砌。

**完整示例：**
> 一张修仙小说封面。一个穿青色道袍的年轻男子盘膝坐在幽暗洞府中闭目修炼，身体周围浮现发光的灵力经脉回路图，经脉图呈现电路板和数据流的可视化风格，蓝色灵力沿着优化后的高效路径快速流转。韩漫Webtoon风格，精致写实人物，深蓝与冰蓝主色调，电影级高对比度光影。画面上方用巨大的浮夸金色立体书法字体分两行写着标题，第一行"学霸修仙"，第二行"从科研开始"，字体要大要醒目，带发光效果和金属质感描边。

## 二、标题文字嵌入

### 2.1 基本规则

- **必须用双引号**包裹标题文字，模型对双引号内文字渲染准确度更高
- 明确分行：写明"第一行""第二行"的具体内容，不要让模型自行断行
- 字体描述放在提示词**最后**，因为信息优先级最高

### 2.2 分行建议

| 标题总字数 | 分行策略 | 示例 |
|-----------|---------|------|
| 4-8 字 | 不分行，一行搞定 | "废柴逆袭" |
| 8-12 字 | 均分两行 | "学霸修仙" / "从科研开始" |
| 12-18 字 | 按语义断行，4-8 字/行 | "杀父之仇我用账本算清" / "顶级审计师降临" |
| 18 字以上 | 不建议直接嵌入，考虑缩短或后处理 |

### 2.3 字体风格描述模板

根据小说题材选择合适的字体风格：

**浮夸霸气型（都市爽文/商战/玄幻）：**
> 用巨大的浮夸金色立体书法字体，字体要大要醒目，带发光效果和金属质感描边

**冷峻悬疑型（悬疑/推理/刑侦）：**
> 用银白色粗体无衬线字体，字体要大要醒目，带冰蓝色发光效果和阴影

**古风仙侠型（修仙/古言/武侠）：**
> 用苍劲有力的毛笔书法字体，字体要大要醒目，带金色描边和淡淡的光晕

**温馨治愈型（日常/校园/甜宠）：**
> 用圆润可爱的手写体，字体要大要醒目，带柔和的白色阴影和淡粉色发光

### 2.4 避坑经验

- 每行控制在 **4-8 个字**，字数越多变形概率越高
- 12 字以上的长标题，AI 渲染可能出现漏字、错字、笔画模糊
- 如果标题很长，考虑只嵌入**精简版标题**（如副标题不放入图片）
- 标题位置默认"画面上方"，如果场景上方有遮挡物，改为"画面下方"

## 三、构图策略

### 3.1 标题与主体的空间分配

核心原则：**标题区和主体区不重叠**。

| 方案 | 适用场景 | 描述方式 |
|------|---------|---------|
| 标题在上，主体在下 | 大部分场景 | "画面上方写着标题...主体在画面中下方" |
| 标题在下，主体在上 | 上方有天空/留白的场景 | "画面下方写着标题...主体在画面上方" |
| 标题在留白区域 | 侧边有大量留白 | "画面左侧留白区域写着标题" |

### 3.2 系列封面差异化

同一组 5 张封面需要通过**不同构图角度**制造视觉差异：

| 技法 | 效果 | 示例 |
|------|------|------|
| 正面特写 | 强调人物气场 | 男子单手撑桌，眼神冷厉看向镜头 |
| 俯瞰全景 | 展示世界观规模 | 俯瞰视角，仙山宗门全景 |
| 背影/剪影 | 营造悬念感 | 男人剪影站在黑暗办公室 |
| 仰视角度 | 塑造压迫感/崇高感 | 仰视法庭中的男人，逆光照射 |
| 侧面构图 | 叙事感/氛围感 | 男女主并肩站在悬崖边 |

### 3.3 不同题材的构图模板

**都市商战/职场：**
- 场景：办公室、摩天大楼、金融区、法庭
- 元素：西装、文件、屏幕数据、城市夜景
- 色调：深蓝+金色、黑+暖黄、灰+银

**修仙/玄幻：**
- 场景：山巅、洞府、仙阁、虚空
- 元素：灵力光环、阵法符文、法宝、飞剑
- 色调：深蓝+冰蓝、深紫+星空、青绿+白金

**悬疑/推理：**
- 场景：暗室、雨夜、档案室、犯罪现场
- 元素：剪影、台灯光、破碎证据、阴影
- 色调：黑+暖黄（noir）、灰+红、深蓝+白

**甜宠/校园：**
- 场景：教室、樱花道、咖啡馆、图书馆
- 元素：阳光、花瓣、微笑、日常道具
- 色调：暖粉+白、浅蓝+米黄、柔光

## 四、风格一致性

### 4.1 统一风格锚点

一组封面需要在提示词中保持以下要素一致：

1. **风格关键词**：每组统一使用同一个风格词，如"韩漫Webtoon风格，精致写实人物"
2. **人物特征**：服装颜色、发型、体型在各张图中保持一致
3. **色调方向**：确定一个主色调组合，各张图在此基础上微调

### 4.2 风格关键词参考

| 风格 | 关键词 | 适用题材 |
|------|--------|---------|
| 韩漫写实 | 韩漫Webtoon风格，精致写实人物 | 通用，人物颜值高 |
| 国风插画 | 国风插画，工笔重彩，仙侠气韵 | 修仙/古言 |
| 日系动漫 | 日系动漫风格，赛璐璐上色 | 校园/轻小说 |
| 欧美写实 | 写实油画风格，古典油画质感 | 西幻/历史 |
| 暗黑哥特 | 暗黑哥特风格，厚涂肌理 | 悬疑/恐怖 |

## 六、实战避坑清单

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 标题出现错字/漏字 | 标题过长（>10字/行） | 缩短标题或分行处理 |
| 标题被场景遮挡 | 标题区和主体重叠 | 明确指定标题位置，描述主体避开该区域 |
| 多张封面人物不像同一个人 | 人物描述不够具体 | 固定人物特征描述：服装颜色、发型、体型 |
| 场景太复杂导致人物模糊 | 场景细节过多 | 减少场景元素数量，聚焦一个核心画面 |
| 色调偏离预期 | 色彩描述不够明确 | 使用具体颜色名而非模糊词（用"深蓝"而非"冷色"） |
| 画面泛白/灰蒙蒙 | 提示词缺乏光影描述 | 加上"高对比度""电影级光影"等描述 |
| 文字太小不醒目 | 未强调字体大小 | 加上"巨大的""字体要大要醒目" |

## 七、生成流程建议

### 7.1 标准流程

1. **确定标题分行方案**（参考 2.2）
2. **选择风格锚点**（参考 4.2）
3. **设计 5 张不同构图的场景**（参考 3.2）
4. **为每张图写完整提示词**（套用第一节的公式）
5. **5 张图并行生成**（节省时间）
6. **检查结果**：标题是否正确、人物是否一致、色调是否统一
7. **不满意的重新生成**，微调提示词后重试

### 7.2 提示词检查清单

生成前逐项检查：

- [ ] 是否以"一张[题材]小说封面"开头，明确用途
- [ ] 主体人物描述是否具体（服装、发型、动作、表情）
- [ ] 场景环境是否清晰（地点、时间、氛围）
- [ ] 风格关键词是否统一（与同组其他封面一致）
- [ ] 色调是否明确（具体颜色名而非模糊描述）
- [ ] 标题文字是否用**双引号**包裹
- [ ] 标题分行是否明确（"第一行""第二行"）
- [ ] 字体风格是否与题材匹配
- [ ] 标题位置是否与主体不冲突
- [ ] 是否为自然语言连贯描述（非关键词堆砌）

## 八、完整提示词模板（可直接替换使用）

```
一张{题材}小说封面。{主体人物}在{场景环境}，{人物动作/状态}。韩漫Webtoon风格，精致写实人物，{主色}与{辅色}主色调，{光影描述}。画面{标题位置}用{字体风格}分{行数}行写着标题，第一行"{标题第一行}"，第二行"{标题第二行}"，字体要大要醒目，带{特效描述}。
```

**替换示例（都市悬疑）：**

> 一张都市悬疑小说封面。一个穿深色风衣的男人独自站在雨夜的巷口，手中拿着一份沾满水渍的文件，雨水顺着他的帽檐滴落。韩漫Webtoon风格，精致写实人物，深灰与暗红主色调，戏剧性侧光与阴影。画面上方用巨大的银白色粗体字体分两行写着标题，第一行"雨夜追凶"，第二行"真相只有一个"，字体要大要醒目，带冰蓝色发光效果和阴影。
