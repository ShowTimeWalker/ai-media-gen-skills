### 1.2 签约流程

- **签约门槛**：长篇满 **2万字** 可申请签约，建议写到 **3万字** 再提交（前3章质量决定签约率）
- **申请机会**：作品在 **2万/5万/8万字** 各有 1 次签约申请机会，共 3 次
- **审核周期**：一般 **7个工作日** 完成，通常 2-3 天有结果（双休日及法定节假日不评估）
- **签约方式**：后台申请（点击「申请签约」按钮）或 编辑内投
- **核心考核指标**：**万字完读率** 是最关键的考核指标；阅读人数低于 1000 可能影响评估

### 1.3 收益体系

**签约类型：**

| 类型 | 说明 |
|------|------|
| **独家分成签约** | 按合同比例获得内外部渠道收益分成（短剧、动态漫、影视、有声、出版、漫画、游戏） |
| **优质保底签约** | 千字 15-3000 元保底/买断；读者打赏作者获 70% |

**福利政策：**

| 福利 | 标准 |
|------|------|
| **普通全勤奖** | 600元/月；作品签约且有效字数满10万字的次月起；日更4000字以上且当月无缺勤；App听读分成 >= 500元 |
| **进阶全勤奖** | 最高3000元/月；日更6000字以上；按当月广告分成的10%计算 |
| **分成激励** | 10-100万字之间额外发放10%-20%分成收益激励，单月最高1万元 |
| **乘风计划** | 全勤奖前提下，所有平台听读收益额外奖励分成15% |
| **收入保障计划** | 优质作品可申请分成转保底（15-150元/千字）；需15-50万字+无连续15天断更 |
| **亿级扶持** | 2025年推出亿级现金扶持优质内容，单人奖金超百万 |

> 注意：同年同作者名下仅一部作品可享受全勤奖

### 1.4 创作节奏建议

| 项目 | 建议 |
|------|------|
| **章节字数** | 每章 **2050-2300字**，有利于有声转化和冲奖励 |
| **日更节奏** | 每天码4000-8000字，正常发布2000-6000，余下存稿 |
| **开书时机** | 月初/月底开文，有利于冲全勤奖 |
| **开头质量** | 重点打磨前3章（"黄金三章"），开头决定编辑第一印象 |
| **开书前几天** | 日更4000-6000字，争取5天内积累良好数据表现 |

### 2.3 修仙书名规律

## 五、修仙末世融合写作建议

### 5.1 可行的融合方向

从当前榜单趋势看，修仙×末世有几种可行模式：

| 模式 | 设定核心 | 爽点来源 |
|------|---------|---------|
| **末世修仙** | 末世后灵气复苏/末世前修仙者穿越 | 以修仙手段碾压末世威胁 |
| **系统修仙+末世求生** | 系统辅助修仙+末世副本 | 双重金手指，兼容两方读者 |
| **长生者在末世** | 长生修仙者在末世苏醒 | 碾压级战力+幽默反差 |
| **诸天万界+末世位面** | 穿越不同末日世界 | 多副本+不同末世体验 |

### 5.2 番茄读者偏好总结

**修仙读者要什么：**
- 凡人逆袭的代入感（灵骨差、杂役弟子、废柴开局）
- 稳定可预期的成长体系（炼气→筑基→金丹→元婴…）
- 金手指的爽感释放（聚宝盆强化、词条叠加、火球术加成万倍）
- 「苟道」与「不争」的反差（表面低调、实则碾压）

**末世读者要什么：**
- 末世前的紧迫感（倒计时、囤物资）
- 与人性的对抗（白莲花、邻居、背叛者）
- 安全感+优越感（安全屋、物资堆积、异能碾压）
- 黑化不圣母（拒绝道德绑架）

### 5.3 章节节奏建议

- **前3章**：金手指觉醒+世界观展开+第一个爽点
- **前10章**：确立主角目标+打脸第一波反派+展现升级潜力
- **前30章**：第一个小高潮+世界观深化+多角色出场
- **后续**：每10-15章一个小副本/高潮，维持追更粘性

### 5.4 书名公式参考
