# 豆包图生图提示词模板

## 通用模板

```text
请用豆包图生图处理这张图片：{本地路径或图片 URL}

保留内容：{人物脸部 / 主体姿态 / 构图 / 产品主体 / 关键文字}
修改内容：{背景 / 服装 / 光线 / 配色 / 风格 / 局部元素}
目标风格：{写实摄影 / 电影感 / 插画 / 电商精修 / 海报风}
画面要求：{镜头感、构图、清晰度、质感}
补充细节：{天空、阴影、材质、边缘细节、反光、层次}
限制条件：{不要改脸 / 不要裁切主体 / 不要新增人物 / 不要改文字}
避免出现：{涂抹感、结构错乱、比例失衡、局部模糊、脏边}
尺寸：{2K 或自定义}
输出文件名：{可选}
```

## 精简模板

```text
用豆包图生图处理 {图片路径或 URL}，保留{必须保留的内容}，把{需要修改的部分}改成{目标效果}，整体风格为{风格}，强调{关键细节}，避免{负面要求}，尺寸 {尺寸}。
```

## 适合图生图的提示词结构

```text
保留{主体与构图}，将{待修改部分}调整为{目标风格/场景}，增强{光线/色彩/材质/氛围}，确保{关键约束}，画面清晰自然，细节统一。
```

## 示例

```text
请用豆包图生图处理这张图片：resources/images/climb2.jpeg

保留内容：人物姿态、人物脸部轮廓、整体构图
修改内容：天空颜色、环境光线、整体氛围
目标风格：日落暖光的电影感写实摄影
画面要求：主体清晰，背景层次更丰富，保留真实山地质感
补充细节：增加金色夕阳侧光和云层层次，增强岩石纹理与空气透视
限制条件：不要改变人物动作，不要裁切主体，不要新增其他人物
避免出现：局部涂抹、人物变形、手部异常、背景脏乱
尺寸：2K
输出文件名：climb_sunset.png
```
