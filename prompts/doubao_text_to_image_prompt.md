# 豆包文生图提示词模板

## 通用模板

```text
请用豆包文生图生成一张图片。

主题：{要生成的主体}
用途：{海报 / 封面 / 头像 / 插图 / 商品图}
风格：{写实摄影 / 电影感 / 插画 / 3D / 国风 / 二次元}
画面描述：{主体外观、动作、场景、构图、镜头角度}
光线氛围：{晨光 / 日落 / 霓虹夜景 / 冷调 / 暖调 / 高对比}
色彩倾向：{主色、辅色、整体色调}
细节要求：{服装、材质、背景元素、表情、清晰度}
避免出现：{低清晰度、模糊、畸形手、多余肢体、脏乱背景、文字错误}
尺寸：{2K 或自定义}
输出文件名：{可选}
```

## 精简模板

```text
用豆包文生图生成一张{风格}的{主题}，场景是{场景}，主体正在{动作}，采用{构图/镜头}，整体氛围{氛围}，色调{色调}，突出{关键细节}，避免{负面要求}，尺寸 {尺寸}。
```

## 高质量提示词结构

```text
{主体}，{主体特征}，{动作/姿态}，{场景}，{构图/镜头}，{光线}，{色彩}，{风格}，{材质与细节}，high detail，clean composition
```

## 示例

```text
请用豆包文生图生成一张图片。

主题：一位年轻女性登山者
用途：品牌海报
风格：电影感写实摄影
画面描述：人物站在山脊上眺望远方，背着专业登山包，远处是雪山和云海，低机位广角构图
光线氛围：日落暖金色侧光，空气通透，有轻微雾气
色彩倾向：暖金色和岩石灰蓝色对比
细节要求：冲锋衣与背包纹理清晰，人物五官自然，岩石层次分明
避免出现：模糊、低清晰度、畸形手、多余肢体、杂乱背景
尺寸：2K
输出文件名：mountain_poster.png
```
