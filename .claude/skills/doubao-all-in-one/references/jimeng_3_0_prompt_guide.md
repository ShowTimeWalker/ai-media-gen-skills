# 即梦 3.0 视频生成 提示词最佳实践

## 模型简介

即梦 3.0 是火山引擎推出的视频生成模型，支持文生视频、首帧图生视频、首尾帧图生视频三种模式，提供 720P 和 1080P 两种分辨率。相比 Seedance 1.5 Pro，即梦 3.0 **不支持音频生成**，专注纯视频画面输出。

### 与 Seedance 1.5 Pro 的关键差异

| 能力 | 即梦 3.0 | Seedance 1.5 Pro |
|------|----------|-------------------|
| 分辨率 | 720P / 1080P | 480P / 720P / 1080P |
| 音频生成 | 不支持 | 支持（环境音/对白/BGM） |
| 宽高比 | 文生视频可选，图生视频由图片决定 | 自由选择 |
| 视频时长 | 5s（121帧）/ 10s（241帧） | 2~12s |
| 参考图 | 首帧/首尾帧 | 首帧/首尾帧 |

---

## 提示词公式

```
主体 + 运动 + 环境（非必须）+ 运镜/切镜（非必须）+ 美学描述（非必须）
```

即梦 3.0 不支持声音描述，提示词应专注于视觉画面。建议中英文均可，不超过 400 字。

## 基础原则

### 1. 描述必要的信息

- **给出主体、运动的限定描述**：描述主体的外貌特征、穿着、动作及运动方式
- **给出画面应体现的直观信息**：描述场景中可见的视觉元素和物理效果
- **善用程度副词**：使用"缓慢"、"有力"、"轻轻"、"快速"等副词控制动作节奏

**示例：**
> 一个面容沧桑、身穿中世纪海盗服饰的男子站在大海边黑色的礁石上。男子的表情富有激情，他向天空有力的举起双手，动作透露出对自由的渴望。

### 2. 描述清晰的信息

- **prompt 与画面形成正确对应**：描述的内容应在画面中有直观体现
- **用特征指定主体**：通过性别/年龄/穿着/动作等特征唯一标识角色，全文保持一致

**示例：**
> 画面最左边赛车服形象的人是赛车手、中间中国青年形象的人是导演、最右边黑人手上抱着摄影机形象的人是摄影师。

### 3. 做精准的切镜描述

- **明确区分每个镜头**：告诉模型确切的切镜信息（"切镜至第二镜头"）
- **精准撰写切镜时机**：在特定动作或对话完成后切镜
- **切镜之间有明确的景别/内容区分**：每个镜头的景别、角度、内容应清晰不同

**示例：**
> 镜头1 正面中景。普通卧室内，夜晚，窗外城市微光透进来。
> 镜头2 切镜到手部特写。蓝白色能量迅速包裹双手。
> 镜头3 切镜男子面部特写。能量沿着颈部与下颌攀升。

## 图生视频专项

### 首帧图生视频

- 提示词应描述**从首帧画面出发的动作和变化**，与首帧图片的内容保持一致
- 首帧图片的长宽比决定视频的长宽比，不需要也无法指定 `aspect_ratio`
- 首帧图片限制：最大 4.7MB，最大 4096×4096，最短边 ≥ 320，长宽比 ≤ 3:1

### 首尾帧图生视频

- 提示词应描述从首帧过渡到尾帧的过程和中间变化
- **尾帧图片的长宽比必须与首帧一致**
- 图片格式支持 JPEG、PNG

## 切镜撰写

### 切镜前后风格一致

> 镜头从两人同框的中景开始，女孩转头看着男孩，自信地笑着说："我们一定能做到！"切镜到男孩的近景，他犹豫地回答："你确定吗？"

### 切镜时机控制

在特效变身、情绪转折等关键节点精准切镜：

> 镜头1：中景，人物抬起手看向自己的手掌。
> 镜头2：切镜到手部特写，手掌周围开始出现微弱的发光粒子，缓慢漂浮。
> 镜头3：切镜到人物脖子特写，光粒向上蔓延。

## 进阶技巧

### 指定参考风格提升美感

通过指定美学参考对象，生成风格鲜明的视频：

> 模仿日剧《小森林》的风格，生成一个女孩在果园中采摘苹果视频。
> 参考迪士尼2D动漫电影的风格，生成一个女孩在果园中采摘苹果视频。

### 用摄影术语提升镜头效果

**1. 视角**

- 摄影机角度：高机位/低机位/俯视/仰视/平视/正扣/正仰
- 叙事视角：过肩视角/主观视角/监控视角/望远镜视角
- 主体角度：正面/正侧/四分之三侧/背面/顶面/底面

**2. 景别**

- 远景/全景/中景/近景/特写
- 规范语法：主体 + 景别（如：左边的男人的近景、红衣女子的半身像）

**3. 运镜**

- 运镜公式：起幅构图描述 + 运镜 + 运镜幅度 + 落幅构图描述
- 运镜方式：推/拉/摇/移/跟/升/降/甩/环绕/旋转/变焦

> 镜头以中景开始，以稳定的滑轨速度缓慢推近，朝他的脸部靠近，最终到达极近距离特写。

### 特效运用

- **精准描述触发时机**：特效开始的条件和瞬间
- **精准描述变身过程**：从起始状态到目标状态的逐步变化
- **精准描述变身后的细节**：完成态的材质、颜色、纹理等

> 猫咪被一团柔软温暖的泡泡光晕包裹，身体逐渐拉长站起。绒毛演化为蓬松的橙色短发，耳朵保留为可爱的猫耳。最终变身为一个拥有猫瞳和猫耳、表情俏皮可爱的动漫风格少女。

## 提示词质量检查清单

在执行即梦视频生成前，对照以下清单检查提示词：

- [ ] **主体明确**：是否清楚描述了主体是谁/什么？外貌、穿着、姿态是否具体？
- [ ] **运动描述**：是否描述了主体的动作和运动方式？是否使用了程度副词控制节奏？
- [ ] **角色特征一致**：多人场景中，每个角色的特征描述是否全文一致、可唯一区分？
- [ ] **切镜清晰**（如有切镜）：是否明确标注了镜头编号？景别/内容是否有区分？切镜时机是否精准？
- [ ] **运镜描述**（如有运镜）：是否符合"起幅 + 运镜 + 幅度 + 落幅"公式？是否使用了专业术语？
- [ ] **环境描述**（如需）：是否描述了光线、天气、氛围等环境信息？
- [ ] **风格参考**（如需）：是否指定了美学风格或参考作品？
- [ ] **无声音描述**：即梦 3.0 不支持音频，是否避免了对话/音效/BGM 相关描述？
- [ ] **图生视频一致性**（如使用参考图）：提示词是否与首帧/尾帧图片内容保持一致？
