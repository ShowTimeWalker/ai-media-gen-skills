# 内容安全与格式自检（本 Skill 内摘要）

> 依据番茄小说网公开的内容安全类说明整理为创作对照清单。**具体规则、示例与更新以平台最新公示为准**；若与平台现行规则冲突，以平台为准。

## 一、平台不允许发布的内容（原则性）

内容不得违反法律法规与公序良俗，包括但不限于：危害国家安全与统一、泄露国家秘密、损害国家尊严与利益、宣扬恐怖主义与极端主义；破坏民族团结与宗教政策、宣扬邪教与封建迷信；危害社会公德（谣言、淫秽色情、赌博、毒品、渲染暴力恐怖等）；侵害未成年人权益；侵害他人名誉、肖像、知识产权、商业秘密等。

## 二、常见审核不通过类型（写作时对照）

### 标题与封面

- 书名或章节名含违规、敏感信息，或与内容明显不符。
- 封面缺必要信息或与书本信息不符；封面素材侵权或违规。

### 正文格式

- 图片异常、全文繁体、大段外文、乱码、不分段、空白章、重复内容、字体混乱、章节乱序等。

### 无关信息

- 正文或图片中出现与故事无关的外链、硬广、恶意吐槽平台等。

### 历史题材边界

- 对历史史实与人物恶意歪曲、抹黑等，可能判定为不适合收录（以平台规则为准）。

### 低俗与色情

- 以色情为目的的性描写诱导、违背伦理的性癖细节、未成年性行为不当描写、性犯罪不当宣扬、低俗窥私与软色情擦边等。

### 未成年人保护

- 宣扬或诱导未成年早恋、性行为、霸凌、烟酒斗殴、违法犯罪；过度渲染自杀倾向；恋童、虐待儿童等。

### 不适合发布与引人不适

- 侵犯他人合法权益；违法有害信息；敏感时政不实与有害信息；极端血腥暴力详述、违伦违法暴力且错误宣扬；等等。

### 公序良俗

- 主线或主角宣扬错误三观、破坏公共秩序、恶意挑拨对立与引战等。

## 三、违规可能后果（摘要）

- 停止推荐、取消或影响创作福利资格、冻结未发放稿酬直至修改合格等。
- 严重违规可下架作品、封禁账号并列入黑名单等（以平台通知为准）。

## 四、写作时的实操自检清单

1. 书名、章节名、简介、封面是否与正文一致且无敏感擦边。
2. 分段是否正常、有无乱码与重复章。
3. 是否出现外链、广告、与剧情无关的吐槽。
4. 涉历史、涉未成年、涉性与暴力段落是否落在平台允许边界内。
5. 是否避免影射现实敏感时政与谣言类表述。
