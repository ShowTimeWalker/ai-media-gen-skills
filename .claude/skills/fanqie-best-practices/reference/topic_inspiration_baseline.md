# 题材与风向基线（归纳，非实时榜单）

> 用于「没方向时」的快速选题参考，**不等于**当前官方榜单。若需严格对齐近期热点，应通过维护同步抓取公开页并记入 `sourced_snapshots_index.md`。

## 男频常见高活跃方向（归纳）

1. 都市高概念：规则类、组织博弈、身份反差、职业线+异常设定。
2. 末世 / 游戏入侵：生存资源、阶段灾变、安全屋与团队。
3. 悬疑灵异：异闻、破局、副本式关卡。
4. 玄幻 / 修仙变体：系统化成长、现代语境、快节奏升级。
5. 历史 / 古代：权谋、军争、考据向与强情节向并存。

## 女频常见高活跃方向（归纳）

1. 现言脑洞：读心、穿书、反套路关系、情绪反转。
2. 古言 / 宅斗宫斗：信息差、主母线、家族博弈。
3. 豪门总裁：强冲突+情感拉扯（注意合规边界）。
4. 无 CP 或大女主：事业线、复仇线、成长线。
5. 种田 / 生活流：经营、治愈、细水长流。

## 使用方式

- 先让用户选「男频 / 女频」与偏好的 1～2 个子方向，再给**一句话设定 + 核心矛盾 + 卖点**，避免空泛题材名堆砌。
