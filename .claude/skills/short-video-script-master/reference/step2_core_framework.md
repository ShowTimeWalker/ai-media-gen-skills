# 阶段二：核心框架

## 目标

基于**STEP 1**收集的创意，生成整部剧的核心框架定调，包括剧集规格、主要剧情、核心人物、主要场景、画风和风格基调。

## 规格确认

首先与用户确认剧集规格，按以下选项引导：

**单集时长**（让用户选择）：
- 1 分钟 / 2 分钟 / 3 分钟 / 4 分钟 / 5 分钟

**总时长**（让用户选择）：
- 60 分钟 / 90 分钟 / 120 分钟 / 150 分钟 / 300 分钟

**集数计算**：集数 = 总时长 ÷ 单集时长，**上限 150 集**。如果计算结果超过 150 集，提示用户调整单集时长或总时长。

**默认建议**：
- 如果用户不确定，建议单集 2 分钟 + 总时长 120 分钟 = 60 集，这是红果中等体量短剧的常见配置
- 短平快试水：1 分钟 × 60 分钟 = 60 集
- 大体量精品：3 分钟 × 300 分钟 = 100 集

## 输出格式

```markdown
# [剧名]

## 剧集规格

| 项目 | 数值 |
|------|------|
| 单集时长 | [X] 分钟 |
| 总时长 | [X] 分钟 |
| 总集数 | [X] 集 |

## 剧情概要

[300-500 字的主要剧情概述。交代世界观、核心冲突、主角的起点和终点。要让读者看完就知道这部剧讲什么、爽点在哪里、凭什么追下去。]

## 核心人物

### [角色名]（[身份/定位]，[年龄]）
- **形象**：[具体外貌描述：体型、五官特征、标志性穿着/配饰、气质特征。要足够具体，为后续 AI 生图提供明确参考。如：身高185cm，剑眉薄唇，常穿黑色长衫，右手腕有一道旧疤，眼神冷漠但笑起来很暖]
- **性格**：[2-3个关键词]
- **核心动机**：[这个角色最想要什么]
- **人物弧线**：[从头到尾的变化]

### [角色名]（[身份/定位]，[年龄]）
- **形象**：[具体外貌描述]
- **性格**：[2-3个关键词]
- **核心动机**：[这个角色最想要什么]
- **人物弧线**：[从头到尾的变化]

[重复以上格式，列出所有主要角色。角色数量根据体量控制：60集以内4-6人，60-100集6-8人，100集以上8-12人]

## 主要场景

| 场景 | 描述 | 出现频率 |
|------|------|---------|
| [场景名] | [环境描述：时代、地点、氛围、标志性视觉元素。如：沈家祠堂——明清风格祠堂，青砖灰瓦，正中悬挂"忠义"匾额，常年烛火摇曳] | 高频/中频/低频 |
| [场景名] | [环境描述] | 高频/中频/低频 |

[列出 4-8 个主要场景，标注每个场景在剧中的出现频率]

## 画风与风格

- **画面风格**：[如：古装写实 / 古装唯美 / 现代都市 / 赛博朋克 / 水墨国风]
- **色调基调**：[如：暗调冷色（权谋悬疑感）/ 暖调柔光（甜宠治愈感）/ 高对比强光（战神热血感）]
- **视觉参考**：[如：类似《庆余年》的古装质感，但更偏暗色调]

## 整剧基调

- **核心情绪**：[主情绪，如：逆袭爽感 / 甜爽互宠 / 虐心反转]
- **节奏特征**：[如：前慢后快，前期布局后期爆发 / 全程高能，每集一个反转]
- **一句话概括**：[用一句话讲清楚这部剧是什么、凭什么吸引人]

---
请确认以上核心框架是否符合你的想法。你可以：
1. 确认，开始逐集创作剧本
2. 调整某个方向（请告诉我你想改什么）
3. 推翻重来，换个方向
```

## 关键规则

- **必须等待用户二次确认**，才能进入阶段三
- 用户选择"调整"时，修改后重新展示框架并再次等待确认
- 用户选择"推翻重来"时，回到阶段一重新收集创意
- 剧情概要控制在 300-500 字，不能只写设定要有剧情走向
- 核心人物的**形象描述必须具体**，避免"长得好看""气质出众"等模糊描述，要给出能为 AI 生图直接使用的视觉特征
- 主要场景要标注出现频率，帮助后续制作规划资源
- 整剧基调要与阶段一确定的核心情绪一致

## 文件输出

用户确认框架后，将内容写入 STEP 1 创建的项目目录中：

**文件路径**：项目根目录下的 `outputs/scripts/<标题>_<yyyyMMdd>/framework.md`（注意：是项目根目录的 outputs，与 doubao-all-in-one 的输出目录同级）

- 直接将输出格式中的全部内容写入文件（不含末尾的确认选项部分）
- 同时在对话中告知用户："核心框架已保存到 `outputs/scripts/<标题>_<yyyyMMdd>/framework.md`（项目根目录下）。"
- 然后进入 **STEP 3**
