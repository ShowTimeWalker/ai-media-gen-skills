# 红果短剧爆款类型图谱

> 本文件供剧本创作大师在引导用户选择类型方向时查阅和引用。数据基于 2025-2026 年红果短剧平台趋势整理。
> 女频占比超 70%，是短剧消费的绝对主力。2026 年红果平台 25 部短剧破亿，女频主导方向。

---

## 一、女频类型图谱

### 1. 穿书+甜宠

| 维度 | 内容 |
|------|------|
| **核心爽点** | 反套路 + 甜爽互动。女主不是按剧本走的工具人，而是意外攻略了隐藏大佬 |
| **爆款公式** | 现代女主穿入小说 → 成为书中的炮灰/恶毒女配/小透明 → 为了活命拼命避开剧情 → 每次求生操作反而撞进男主视线 → 意外改写剧情收获爱情 |
| **目标受众** | 18-35 岁女性，偏好轻松、反套路、有脑的甜宠 |
| **情绪节奏** | 恐慌（穿书了怎么办）→ 搞笑（求生操作翻车）→ 心动（意外互动）→ 甜爽升级 → 大结局高甜 |
| **代表作** | 《攻略苗疆少主》《糟糕，和闺蜜一起穿书后把反派玩儿坏了》《少夫人来自东北》 |
| **关键设定要素** | 穿入的书的世界观要有新鲜感（苗疆、仙侠、民国等）、女主的反套路人设（直爽/毒舌/社畜心态）、男主的隐藏身份层次 |
| **避坑** | 不要让女主变成"等着被拯救"的花瓶，2026年趋势是清醒独立的女主更受欢迎 |

### 2. 甜宠+霸总（新派）

| 维度 | 内容 |
|------|------|
| **核心爽点** | 极致的偏爱和独宠，"全世界只有他对你好" |
| **爆款公式** | 女主处于困境 → 男主强势介入（表面冷漠实则深情）→ 身份反差逐步揭开 → 护短名场面 → 甜宠升级 |
| **目标受众** | 18-30 岁女性，偏好幻想型浪漫 |
| **情绪节奏** | 甜度递增，穿插轻度误会后快速和解（不要长虐），整体基调温暖治愈 |
| **代表作** | 红果甜宠类 TOP 榜单作品 |
| **关键设定要素** | 男主的"双面性"（对外冷漠/对女主独宠的反差）、女主不是白莲花（有自己的生活态度）、宠的方式要具体不油腻 |
| **避坑** | 2024 年广电已对"土味霸总"发管理提示，避免强取豪夺、物化女性等旧套路。新派霸总去土味化、重情感连接是方向 |

### 3. 重生+复仇

| 维度 | 内容 |
|------|------|
| **核心爽点** | 前世被辜负→重生清醒→步步为营→让渣男恶人付出代价 |
| **爆款公式** | 前世被最信任的人背叛伤害 → 重生回到悲剧发生前 → 利用前世记忆提前布局 → 逐一清算 → 意外收获真正的爱情 |
| **目标受众** | 25-40 岁女性，偏好爽快、解气、女性觉醒 |
| **情绪节奏** | 虐→清醒→布局→打脸→收获幸福。前期用前世的虐建立动机，中后期集中爽 |
| **代表作** | 红果复仇类热门作品 |
| **关键设定要素** | 前世被背叛的惨烈程度（越惨复仇越爽）、重生后的冷静与狠劲、复仇过程中的意外感情线 |
| **避坑** | 复仇要有策略感，不能只是"运气好"，女性觉醒的主题要让观众共鸣 |

### 4. 大女主/女王叙事（2026 爆款赛道）

| 维度 | 内容 |
|------|------|
| **核心爽点** | 扮猪吃虎——因外貌/身份被小觑，最后狠狠打脸。女主是全剧天花板，绝对不吃瘪 |
| **爆款公式** | 女主以弱势身份出场 → 遭遇不公和轻视 → 逐步展露真实实力 → 碾压所有质疑者 → 收获事业与爱情 |
| **目标受众** | 18-35 岁女性，偏好力量感、独立搞事业 |
| **情绪节奏** | 被小觑（压抑）→ 小试牛刀（惊讶）→ 实力逐层揭露（爽感递增）→ 终极碾压（极致爽感） |
| **代表作** | 《夫人她专治不服》《好一个乖乖女》（白切黑女主+忠犬霸总） |
| **关键设定要素** | 女主的"白切黑"人设（表面温和实则狠辣）、扮猪吃虎的层次感（不能一次全揭完）、周围人的态度反差（从轻视到震惊） |
| **避坑** | 去神化——拒绝完美大女主，要有真实的人性弱点和成长弧光。不要依赖"霸总帮忙"，女主凭自身力量解决问题 |

### 5. 双女主/女性群像（2026 上升赛道）

| 维度 | 内容 |
|------|------|
| **核心爽点** | 女性之间的互助、羁绊与化学反应，脱离三角恋/婚姻俗套，角色关系大胆多元 |
| **爆款公式** | 两个（或多个）性格迥异的女性相遇 → 从对立/误解到理解/联手 → 面对外部挑战共同成长 → 互相成就 |
| **目标受众** | 20-35 岁女性，偏好丰富女性关系刻画、厌倦传统言情套路 |
| **情绪节奏** | 碰撞（火花）→ 并肩（热血/感动）→ 考验（虐心）→ 共赢（高燃/高甜） |
| **代表作** | 红果《弄潮》（女性群像出圈）、双女主短剧在海外百强榜表现亮眼 |
| **关键设定要素** | 女主之间的性格反差（互补/对立）、共同面对的外部压力、各自独立的成长线 |
| **避坑** | 不要把双女主写成"两个女人抢一个男人"，关系核心是女性之间的互助与成长 |

### 6. 闪婚+先婚后爱

| 维度 | 内容 |
|------|------|
| **核心爽点** | 从陌生人到真爱的过程，日常互动中的心动瞬间 |
| **爆款公式** | 意外闪婚（各取所需/长辈安排/阴差阳错）→ 同居/相处中的尴尬和碰撞 → 发现对方的闪光点 → 真心动 → 外部考验 → HE |
| **目标受众** | 25-40 岁女性，偏好生活化、有代入感的爱情 |
| **情绪节奏** | 尴尬搞笑 → 慢慢心动 → 甜蜜日常 → 小虐（外部冲突）→ 甜蜜 HE |
| **代表作** | 《少夫人来自东北》（带喜剧色彩的先婚后爱） |
| **关键设定要素** | 闪婚的理由要有说服力、相处过程中的细节要真实可爱、男女主的性格反差产生化学反应 |
| **避坑** | 不要为了虐而虐，这类类型的观众看的就是甜和温馨 |

### 7. 虐恋+追妻

| 维度 | 内容 |
|------|------|
| **核心爽点** | 前期虐心虐肝 + 后期男主追妻火葬场。观众的情绪在"虐"和"甜"之间反复横跳 |
| **爆款公式** | 男主冷落/误解女主 → 女主默默承受 → 临界点女主彻底离开/清醒 → 男主追悔莫及拼命挽回 → HE 或 BE |
| **目标受众** | 20-35 岁女性，偏好强情绪冲击 |
| **情绪节奏** | 虐→虐→虐→转折（女主清醒）→追妻→HE。关键转折点是女主"不再忍了"的那一刻 |
| **代表作** | 红果虐恋类热门作品 |
| **关键设定要素** | 误解的原因要合理（不能是"女主不解释"这种弱逻辑）、女主离开的方式要干脆、男主追妻的方式要具体有诚意 |
| **避坑** | 2026 年趋势是"清醒虐恋"——女主离开不是因为赌气，而是真的看透了。避免"男主道个歉女主就回去"的降智操作 |

### 8. 真假千金

| 维度 | 内容 |
|------|------|
| **核心爽点** | 身份错位 → 真相大白 → 打脸势利眼。经久不衰的头部套路，适合多元素叠加 |
| **爆款公式** | 出生被调包 → 真千金在穷苦环境励志成长 → 回到豪门面对偏见 → 用实力打脸嫌贫爱富的家人 → 收获真正的归属与爱情 |
| **目标受众** | 18-35 岁女性，偏好逆袭打脸、身份反转 |
| **情绪节奏** | 委屈（被冷待）→ 隐忍成长 → 真相揭露（爽感爆发）→ 持续打脸 → HE |
| **代表作** | 《真千金她是学霸》 |
| **关键设定要素** | 真假千金的性格反差、真千金的成长过程要具体可感、打脸场景要密集且有层次 |
| **避坑** | 当前流行的变体已经不限于单纯真假千金，建议叠加穿书/系统/偷听心声等设定增加新鲜感 |

### 9. 萌宝/带娃甜宠

| 维度 | 内容 |
|------|------|
| **核心爽点** | 萌娃助攻 + 甜宠升级，"孩子比大人更会撩"的反差萌 |
| **爆款公式** | 女主带娃/意外有娃 → 孩子与男主产生羁绊 → 通过萌娃推动男女主感情 → 甜蜜日常 + 萌娃名场面 → HE |
| **目标受众** | 25-40 岁女性，偏好温馨治愈、家庭感 |
| **情绪节奏** | 轻松搞笑 → 萌娃助攻心动 → 甜蜜升温 → 小虐 → 温馨 HE |
| **代表作** | 《好孕甜妻被钻石老公宠上天》《好一个乖乖女》 |
| **关键设定要素** | 萌娃要有鲜明的性格标签（话痨/腹黑/高冷反差）、萌娃推动感情的桥段要自然不刻意 |
| **避坑** | 孩子不能沦为工具人，要有自己的情感线；避免"未婚先孕"等敏感设定 |

### 10. 种田/生活流

| 维度 | 内容 |
|------|------|
| **核心爽点** | 从零开始的经营养成 + 治愈日常 + 逆袭翻身。打破"快节奏打脸"的刻板印象 |
| **爆款公式** | 穿越或重生到资源匮乏的环境 → 用现代知识/技能从零开始 → 逐步改善生活 → 打破偏见 → 收获事业与爱情 |
| **目标受众** | 25-45 岁女性，偏好治愈、生活化、养成感 |
| **情绪节奏** | 困难（代入感）→ 动手解决（满足感）→ 逐步改善（成就感）→ 翻身打脸（爽感）→ 治愈 HE |
| **代表作** | 红果种田类热门作品、"系统+种田"组合 |
| **关键设定要素** | 种田/经营的具体过程要有真实感、从困境到改善的过程要可感可叹、种田与爽点的结合要自然 |
| **避坑** | 不要让种田变成流水账，要有矛盾冲突和情感推进。可叠加系统设定增加趣味性 |

### 11. 治愈救赎

| 维度 | 内容 |
|------|------|
| **核心爽点** | 相互治愈、双向奔赴。满足用户深层情感需求 |
| **爆款公式** | 两个各自受伤的人相遇 → 从试探到信任 → 互相成为对方的光 → 面对过去创伤 → 携手前行 → HE |
| **目标受众** | 20-35 岁女性，偏好深度情感共鸣、治愈系 |
| **情绪节奏** | 疏离 → 慢慢靠近 → 温暖治愈 → 揭开伤疤（虐心）→ 彼此救赎 → 温馨 HE |
| **代表作** | 红果治愈类热门作品 |
| **关键设定要素** | 双方的创伤要有真实感、治愈过程要具体细腻、不能只是"爱情解决一切" |
| **避坑** | 治愈不是糖水，要有深度和张力。避免悬浮的"完美救赎" |

### 12. 女尊世界观（蓝海赛道）

| 维度 | 内容 |
|------|------|
| **核心爽点** | 权力反转——女尊男卑的社会结构，满足女性观众对权力叙事和新型两性关系的想象 |
| **爆款公式** | 女主进入或觉醒于女尊世界 → 享受权力主导地位 → 与处于"弱势"的男性产生反套路互动 → 面对权力挑战 → 确立地位 + 爱情 |
| **目标受众** | 18-30 岁女性，偏好新鲜设定、权力叙事、反套路 |
| **情绪节奏** | 新鲜感（世界观冲击）→ 有趣（权力反转的日常）→ 心动（反套路恋爱）→ 考验 → HE |
| **关键设定要素** | 世界观要自洽、权力反转要落实到细节（日常互动/社会规则）、男性角色的人设反差（如强势外表+忠犬内核） |
| **避坑** | 蓝海赛道尚未出现现象级爆款，世界观设定要经得起推敲，不能只是"换个性别演霸总" |

### 13. 末世+大女主（2026 上升趋势）

| 维度 | 内容 |
|------|------|
| **核心爽点** | 极端环境下的生存智慧 + 女性力量觉醒，从被保护者变成团队核心 |
| **爆款公式** | 灾难降临（丧尸/异变/自然灾害）→ 社会秩序崩塌 → 女主从普通人觉醒异能/获得系统 → 组建幸存者团队 → 资源争夺 + 人性考验 → 逐步揭开末世真相 → 建立新秩序 |
| **目标受众** | 18-30 岁女性，偏好强情节、生存感、女性力量 |
| **情绪节奏** | 恐慌（灾难降临）→ 绝境求生（紧张刺激）→ 觉醒（第一次展现能力）→ 团队羁绊（热血感动）→ 真相揭露（震撼）→ 重建希望（燃） |
| **代表作** | 末世囤物资类、末世系统类短剧 |
| **关键设定要素** | 末世类型要明确（丧尸/异兽/天灾/病毒）、物资管理要有策略感、人性考验要真实不悬浮、女主的成长要有过程不能一步登天 |
| **避坑** | 末世设定要有规则感，不能无限开挂；避免纯打斗流水账，要有人物情感和人性深度；注意红果审核尺度，丧尸/血腥元素需控制在平台允许范围内 |

### 14. 末世+双女主（蓝海组合）

| 维度 | 内容 |
|------|------|
| **核心爽点** | 末世环境下两个女性的互补求生 + 羁绊深化，脱离"等男人来救"的套路 |
| **爆款公式** | 两个性格迥异的女性在灾难中相遇 → 一个武力值高/一个智商高，互补生存 → 从不信任到彼此依靠 → 共同面对外部威胁和人性考验 → 互相成就 |
| **目标受众** | 18-30 岁女性，偏好双女主、末世生存、女性互助 |
| **情绪节奏** | 末日冲击 → 碰撞磨合 → 并肩求生 → 信任危机 → 绝境翻盘 → 双赢 |
| **关键设定要素** | 两位女主的能力互补要有说服力、末世环境要有生存压力感、外部威胁要持续存在 |
| **避坑** | 不要写成"末世谈恋爱"，生存压力才是核心驱动力 |

## 二、男频类型图谱

### 1. 重生+权谋

| 维度 | 内容 |
|------|------|
| **核心爽点** | 利用前世信息差降维打击，一步步瓦解敌人的布局 |
| **爆款公式** | 惨死/被害 → 重生回到关键时间节点 → 隐忍蛰伏 → 逐步揭开前世真相 → 关键时刻一击翻盘 |
| **目标受众** | 25-40 岁男性，偏好智斗、逆袭、权力博弈 |
| **情绪节奏** | 压抑蓄力 → 小胜积攒信心 → 真相逐步揭开 → 大爆发。前期克制，后期释放，爽感递增 |
| **代表作** | 《一品布衣》《十八岁太奶奶驾到》系列 |
| **关键设定要素** | 重生时间节点（越早越难但越爽）、敌我身份反差（弱者 vs 强者）、前世记忆的具体应用场景 |
| **避坑** | 不要让主角开局就无敌，要有过程感；权谋要有逻辑，不能只是"主角说什么都对" |

### 2. 战神+家国

| 维度 | 内容 |
|------|------|
| **核心爽点** | 隐藏身份曝光的瞬间碾压感 + 保家卫国的热血使命感 |
| **爆款公式** | 被人看不起的"废物" → 实际上是战功赫赫的隐藏大佬 → 身份曝光震惊所有人 → 对外保家卫国、对内清算仇敌 |
| **目标受众** | 20-35 岁男性，偏好热血、军旅、身份反转 |
| **情绪节奏** | 被轻视（憋屈）→ 小试牛刀（惊讶）→ 身份层层揭露（震撼）→ 大场面碾压（极致爽感） |
| **代表作** | 各类"战神归来""隐身侍卫"系列 |
| **关键设定要素** | 隐藏身份的层次感（不能一次全揭完）、反派的嚣张程度（越嚣张翻盘越爽）、家国情怀的融入节点 |
| **避坑** | 避免纯碾压无逻辑，2026 年趋势是融入智斗和家国大义，不能只剩"打脸" |

### 3. 神豪逆袭

| 维度 | 内容 |
|------|------|
| **核心爽点** | 突然获得巨额财富后的消费碾压和社会地位翻转 |
| **爆款公式** | 被人看不起的穷人/小人物 → 获得系统/继承/中奖 → 用钱碾压曾经看不起自己的人 → 发现金钱背后的更大格局 |
| **目标受众** | 18-30 岁男性，偏好爽快直给的剧情 |
| **情绪节奏** | 被羞辱 → 系统激活 → 第一次花钱爽 → 持续升级碾压 → 情感/价值观升华 |
| **代表作** | 红果神豪类 TOP 榜单作品 |

| **避坑** | 不能只是"花钱流水账"，要有人物成长和价值观转变 |

### 4. 系统流

| 维度 | 内容 |
|------|------|
| **核心爽点** | 系统任务驱动下的"升级打怪"，每完成一个任务获得奖励 |

| **代表作** | 快穿执行者系列、系统穿越类 |
| **关键设定要素** | 系统的规则和限制（有限制才有策略感）、任务的多样性和趣味性、系统与主角的关系演变 |
| **避坑** | 任务不能太水，每个任务要有情感或戏剧价值 |

### 5. 年代文重生

| 维度 | 内容 |

## 四、红果平台规范

### 内容结构铁律

**前三秒决定生死**：
- 开场必须在 3 秒内制造冲击——视觉冲击、情绪冲击、或认知冲击
- 绝对不能在开头慢热铺垫

**高频爆点节奏**：
- 每 30-60 秒至少一个情绪爆点（反转、打脸、真相揭露、名场面）
- 观众没有耐心等超过 1 分钟的平淡

**黄金四段式**：

```
Hook（0-3秒）→ 悬念/冲突建立（3-10秒）→ 核心内容/情绪推进（10-45秒）→ 结尾钩子（最后3秒）
```

- Hook：抓住注意力
- 悬念：让观众想知道"然后呢"
- 核心：情绪推进，一个或多个转折
- 结尾钩子：留下悬念或情感冲击，驱动看下一集

### 付费转化节奏

- 前 3 集完成核心冲突建立 + 身份悬念抛出
- 第 8-12 集为关键付费转化节点
- 爆款完播率通常在 25% 以上

## 五、爆款共性提炼

### 四条铁律

1. **极致情绪驱动**：爽、甜、虐、燃——选一个主情绪打透，不要什么都想要
2. **人物标签化**：每个角色一眼就能记住（"阴狠嫡兄""直爽穿书女""白切黑乖乖女"），避免面目模糊
3. **冲突高频密集**：不拖沓，每段对话都要推进矛盾或揭示信息，没有"过场戏"
4. **开头即高潮**：第一句台词/第一个画面就要有冲击力，不要从"很久很久以前"开始

### 女频 vs 男频爽点差异

| 维度 | 女频 | 男频 |
