## 二、可直接开写的“高潜题材模板”

### 男频模板

- 模板 A：`高概念规则 + 小人物破局 + 组织对抗`
  - 钩子句：全城都在遵守那条新规则，只有他发现“规则会撒谎”。

- 模板 B：`末世入侵 + 资源经营 + 阶段性灾变`
  - 钩子句：第二次灾变倒计时只剩七天，而他的安全屋里还缺一个关键人。

- 模板 C：`现代修仙 + 职场映射 + 团队升级`
  - 钩子句：公司晨会改成了“灵压考核”，迟到的人会被扣寿元。

### 女频模板

- 模板 D：`无CP事业线 + 高压成长 + 反控制`
  - 钩子句：她重生后做的第一件事，不是复仇，而是先切断所有情绪绑架。

- 模板 E：`现言脑洞 + 反套路恋爱博弈 + 自我觉醒`
  - 钩子句：她突然能听见反派心声，才知道自己一直活在别人写好的剧本里。

- 模板 F：`古言宅斗 + 信息差 + 身份跃迁`
  - 钩子句：主母手里的账本只少了一页，却足够让半个侯府陪葬。
