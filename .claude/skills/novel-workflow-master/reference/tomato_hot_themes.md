## 二、可直接开写的"高潜题材模板"

### 修仙方向

- **模板 A**：`凡人流 + 单一金手指 + 稳定升级`
  - 参考：《聚宝仙盆》（东方仙侠榜首，352万字）、《凡骨》（787万字）
  - 钩子句：杂役弟子偶然获得一个能强化一切的盆，下品丹药放进盆里，一日之后变成两颗极品。
  - 核心爽点：金手指简单直观（强化一切），成长体系清晰（丹→功法→法宝→草药→灵石）

- **模板 B**：`凡骨逆天 + 无系统 + 热血逆袭`
  - 参考：《凡骨》（东方仙侠榜第2，787万字）
  - 钩子句：世间灵骨共分四品，余者皆为凡骨，无缘修行。一介凡骨许太平，誓要登仙！
  - 核心爽点：无系统设定增强代入感，"凡骨也能斩妖除魔登仙"的逆袭宣言强钩子

- **模板 C**：`长生苟道 + 时间跨度 + 轻松诙谐`
  - 参考：《开局长生万古，苟到天荒地老》（玄幻脑洞榜首，606万字，已出版）
  - 钩子句：穿越修仙界，觉醒长生系统，我只想带着老牛看遍世间繁华。一个不经意，他露出了腰间的三把开山斧。
  - 核心爽点：苟道反差（表面不争→被迫出手碾压）、时间跨度感（历经多个修仙大时代）、轻松幽默

- **模板 D**：`传统玄幻 + 腹黑主角 + 扮猪吃虎`
  - 参考：《惊鸿》（传统玄幻榜首，858万字）、《天渊》（551万字）
  - 钩子句：烟雨楼背后还有一人，方才是真正的创始人，以纨绔子弟的身份为掩饰，拨弄风云，布局天下！
  - 核心爽点：身份伪装+暗中操盘、腹黑反差、非后宫

- **模板 E**：`洪荒题材 + 特殊身份 + 碾压流`
  - 参考：新书榜洪荒题材爆发（洪荒：功法小龙龙、洪荒：伏羲大帝等）
  - 钩子句：穿越洪荒，他发现自己的系统画饼才是真的——说一声"天道酬勤"，功德真到账了。
  - 核心爽点：洪荒世界观认知度高、开挂身份（盘古之弟/伏羲转世）、简单粗暴爽感

### 末世方向

- **模板 F**：`重生 + 囤货 + 黑化不圣母`
  - 参考：《冰河末世，我囤积了百亿物资》（科幻末世榜第2，634万字）
  - 钩子句：全球进入冰河时代，95%的人类丧生。上一世心地善良被帮助过的人杀死，重生回末世前一个月，觉醒空间异能。
  - 核心爽点：前世背叛→今生黑化、疯狂囤物资的安全感、白莲花打脸

- **模板 G**：`多副本末日 + 成长主线 + 群像`
  - 参考：《末日生存方案供应商》（科幻末世榜首，228万字）
  - 钩子句：万千世界濒临灭亡，唯有挽救客户才能存续文明火种。丧尸围城、地涌魔泉、草木狂潮、太阳爆发……"我从不失手。"
  - 核心爽点：多末日类型轮换（不单调）、客户互动有趣、非无脑爽

- **模板 H**：`时间暂停 + 末世 + 搞笑碾压`
  - 参考：《时间静止，邪神都给我跪下！》（阅读榜热门）
  - 钩子句：他能让时间静止，而且每天十分钟，还会累积叠加。废土统领跪在地上求饶，他却心痛：浪费了整整三秒！
  - 核心爽点：时间暂停能力简单直观、搞笑反差（惜时如命到离谱）、轻松碾压

### 修仙×末世融合

- **模板 I**：`灵气复苏 + 修仙碾压末世`
  - 钩子句：末世第三年，所有人都还在和丧尸搏命，他刚突破筑基期。别人用枪，他用剑。

- **模板 J**：`长生者在末世苏醒 + 幽默反差`
  - 钩子句：沉睡万年的修仙者醒来，发现世界已经变成废土。他揉了揉眼睛，拔出了腰间的开山斧。

## 四、简介模板参考（真实热门书结构）

### 修仙简介

**凡人流模板：**
```
【标签1】【标签2】身份描述【主角名】获得【金手指】。
金手指具体效果展示（1-2句）。
主角从此走上强者巅峰……
```

**传统玄幻模板：**
```
一句话设定背景/困境。
主角面临的具体压力（悔婚/欺压/修为尽废）。
反转/悬念收尾。
```

**脑洞搞笑模板：**
```
【已出版/标签堆叠】入坑谨慎。
大设定介绍（1-2句）。
主角性格+反差搞笑场景（2-3句）。
口头禅/标志性台词收尾。
```

### 末世简介

**重生囤货模板：**
```
末世+重生+核心设定堆叠
末世背景（全球XX%丧生！）
上一世悲剧+重生回X天前
囤物资/造安全屋的爽点
白莲花/富二代打脸对话
主角冷漠宣言收尾
```

**多副本模板：**
```
主角身份+困境背景
被神秘组织录用
万千世界末日设定（3-5种）
客户对话（困境-能力-信任）
标志性台词收尾
```
