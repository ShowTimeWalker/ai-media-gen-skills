# 阶段二：规格与结构骨架

## 目标

把阶段一框架转化为可执行的结构蓝图，明确体量、卷结构和节奏曲线，输出 `blueprint.md`。

## 输入

- `framework.md`

## 输出

- `blueprint.md`

## 规格模板（默认）

| 规格 | 章节数 | 单章字数建议 | 总字数建议 |
|------|--------|--------------|------------|
| 短篇 | 5-20 章 | 1200-2200 | 1-5 万 |
| 中篇 | 20-50 章 | 1500-2600 | 5-15 万 |
| 长篇（标准） | 60-120 章 | 1800-3200 | 15-40 万 |
| 长篇（扩展） | 120-240 章 | 2000-3500 | 40-80 万 |
| 超长篇（上限） | 250-500 章 | 2000-4000 | 80-100 万 |

用户可自定义，优先以用户要求为准。

## 门禁规则

- 若章节数、卷结构、节奏曲线任一项未确认，不进入阶段三。
- 阶段二完成后请求确认：

