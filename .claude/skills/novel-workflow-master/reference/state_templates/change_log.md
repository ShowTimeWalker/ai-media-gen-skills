# 变更日志模板（state/change_log.md）

> 说明：每章写作后记录“状态变更”，作为追责与回溯依据。

## 字段定义

- 记录ID：`LOG_001`
- 章节：`chXX`
- 改动对象ID：`CHAR_*` / `EVT_*` / `ARC_*` / `FORESHADOW_*` / `RULE_*`
- 改动前：变更前状态摘要
- 改动后：变更后状态摘要
- 原因：为什么发生此变更
- 引用片段：正文中的锚点句或段落摘要
