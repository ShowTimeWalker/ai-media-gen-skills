# 规则日志模板（state/rule_log.md）

> 说明：用于维护世界规则与例外，规则 ID 使用 `RULE_*`。

## 字段定义

- 规则ID：`RULE_001`
- 规则描述：规则内容
- 首次出现章节：规则首次明确出现的章节
- 例外：允许的破例条件
- 代价：触发例外时的成本或后果
- 当前状态：有效/待确认/废弃
