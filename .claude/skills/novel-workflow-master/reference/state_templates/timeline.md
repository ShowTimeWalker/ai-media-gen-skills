# 时间线模板（state/timeline.md）

> 说明：按事件发生顺序维护，不按写作顺序维护。事件 ID 使用 `EVT_*`。

## 字段定义

- 事件ID：`EVT_001`
- 章节：`ch01`
- 发生时间：世界内时间（如“元历 312 年 春”）
- 地点：事件主要发生地点
- 参与角色：`CHAR_*` 列表
- 结果：事件直接结果
- 影响范围：对后续章节/支线的影响
