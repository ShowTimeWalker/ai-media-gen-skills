# 故事图谱模板（state/plot_graph.md）

> 说明：统一记录主线、支线、伏笔的推进状态。主线/支线 ID 用 `ARC_*`，伏笔 ID 用 `FORESHADOW_*`。

## 字段定义

- 线索ID：`ARC_MAIN_001` 或 `FORESHADOW_001`
- 类型：主线/支线/伏笔
- 当前状态：未启动/推进中/已回收/中断
- 埋点章节：首次出现章节
- 预期回收章节：计划回收章节
- 实际回收章节：真实回收章节（未回收可空）
- 关联角色：`CHAR_*` 列表

## 记录表

| 线索ID | 类型 | 当前状态 | 埋点章节 | 预期回收章节 | 实际回收章节 | 关联角色 |
|--------|------|----------|----------|--------------|--------------|----------|
| ARC_MAIN_001 | 主线 | 推进中 | ch01 | ch30 |  | CHAR_001 |
| FORESHADOW_001 | 伏笔 | 未启动 | ch02 | ch12 |  | CHAR_002 |
