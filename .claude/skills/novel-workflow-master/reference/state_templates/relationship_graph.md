# 关系图谱模板（state/relationship_graph.md）

> 说明：记录角色关系的当前状态与最近变更。角色 ID 用 `CHAR_*`，变更事件用 `EVT_*`。

## 字段定义

- 关系ID：`REL_001`
- 角色A：`CHAR_*`
- 角色B：`CHAR_*`
- 关系类型：亲情/师徒/敌对/盟友/恋爱等
- 强度：-5 到 +5（敌对到亲密）
- 最近变更事件ID：`EVT_*`
- 备注：关系变化原因
