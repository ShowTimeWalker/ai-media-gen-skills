# 人物状态模板（state/character_state.md）

> 说明：每章写后更新关键角色当前状态。角色 ID 使用 `CHAR_*`。

## 字段定义

- 角色ID：`CHAR_001`
- 角色名：姓名或代号
- 当前目标：此时角色最优先目标
- 秘密/误解：角色隐藏信息或认知偏差
- 关系变化：与谁发生了关系变化（引用 `CHAR_*`）
- 资源/伤病：能力、物资、伤势等状态
- 风险：角色当前最大风险
